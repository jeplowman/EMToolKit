EMToolKit.algorithms.multi_inst_dem_wrapper module

EMToolKit.algorithms.multi_inst_dem_wrapper Module

This module provides a wrapper for Differential Emission Measure (DEM) analysis using multiple instruments. It includes functions for reprojecting data with uncertainties and plotting the results.

Functions:

multi_inst_simple_dem_wrapper: Wraps simple DEM analysis for multiple instruments. reproject_with_uncertainties: Reprojects a sequence of maps to the smallest map. plot_reprojected: Plots reprojected maps and their uncertainties.
